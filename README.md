# htopmini by jNizM
Small System-Info Tool via AutoHotkey

  
## Features
* Weather: Location & Temperature
* Memory: RAM (Used, Free, Total)
* HDD: Fixed, Removable & Network Drives (Used, Free, Total)
* NetTraffic: IN & OUT (dont work 100%)
* Clear Memory
